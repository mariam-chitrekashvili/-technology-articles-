# -technology-articles-

📌 Description

This project is a simple technology blog layout created using HTML5 and CSS3.
It displays a section of technology articles, where each article includes an image, a title, and a publication date.

The goal of this project was not to build something complex, but to focus on clean structure, proper alignment, and visual balance. It is a static layout built without any frameworks or JavaScript, allowing me to concentrate fully on core front-end fundamentals.

📚 What I Learned

While working on this project, I practiced and improved:

Writing clean and semantic HTML structure

Organizing content in a clear and logical way

Using Flexbox for layout and alignment

Controlling spacing, margins, and padding properly

Creating consistent design and visual hierarchy

Structuring CSS in a more readable and maintainable way

This project helped me rebuild my confidence in front-end basics and reminded me that strong fundamentals are essential before moving to more advanced technologies.
